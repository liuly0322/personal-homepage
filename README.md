Make sure you have [Node.js](https://nodejs.org) and [yarn](https://yarnpkg.com) installed.

```bash
# Clone the repo
git clone ...
cd ...

# Install VuePress
yarn global add vuepress

# Install dependencies
yarn

# Run
$ yarn dev
```

Check your site at `http://127.0.0.1:8080/`
